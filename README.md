## Dataset

TempNY.csv

## Objetivo

Pronosticar temperaturas futuras de la ciudad de Nueva York mediante modelado ARIMA(p.d.q)

## Técincas utilizadas

Modelo ARIMA

Prueba Dickey-Fuller

Comparaciones ARIMA-AIC

Ajustes de Modelo

## Resultados

Métricas de error:
MAE  = 1.1366
RMSE = 1.3462
MAPE = 2.01%

Al tener un MAPE de 2.01% nos indica que el modelo es bastante confiable para realizar predicciones de temperaturas futuras.
